## How It Works

**Manifests** = Memory architecture (JSON files, human-readable, exportable)

- Core identity, context, emotional weights, system rules
- Modules load on-demand (family, music, projects, people, business)
- Everything cumulative, nothing resets, cross-platform compatible

**Right now:** Manual (load manifests each session)

**Future:** Self-hosted, API-driven, voice synthesis, avatar rendering
